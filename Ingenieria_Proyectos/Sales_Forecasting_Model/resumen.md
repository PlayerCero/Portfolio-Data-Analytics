# ForeCast-Analytics: Análisis de Series de Tiempo y Predicción de Demanda 📈

## 📋 Resumen del Proyecto
Este proyecto se centra en el desarrollo de un modelo estadístico avanzado para el análisis y la previsión de ventas mensuales. Utilizando técnicas de descomposición de series de tiempo, se identificaron los factores subyacentes que afectan el rendimiento comercial, permitiendo generar pronósticos con alta precisión para la toma de decisiones estratégicas.

## 🔍 Análisis de Componentes
* **Análisis Estacional (S)**: Identificación de patrones estacionales donde se determinó que meses como **Diciembre favorecen las ventas en un 6.82%**, mientras que Febrero presenta una contracción del 4.71%.
* **Índice Ciclo-Económico (C)**: Evaluación del impacto de la economía externa en las ventas, identificando variaciones significativas en el primer, tercer y cuarto año de operación.
* **Modelado de Tendencia (T)**: Comparación técnica entre funciones de crecimiento **Lineal** y **Exponencial**, determinando la mejor curva de ajuste para la proyección a largo plazo.

## 🚀 Metodología de Pronóstico
* **Suavizamiento y Promedios Móviles**: Aplicación de promedios móviles (MM12 y MM17) para eliminar el ruido y resaltar la tendencia real.
* **Previsión con Límites de Confianza**: Generación de escenarios de ventas para el próximo año, incluyendo límites superiores e inferiores para la gestión de riesgos de inventario.
* **Validación de Modelos**: Uso de herramientas de análisis de datos para comparar la previsión estadística frente a resultados reales (Prev Excel vs Pronostico).

## 🛠️ Herramientas y Habilidades
* **Estadística Descriptiva e Inferencial**: Dominio de componentes T, S, C e I.
* **Modelado Predictivo**: Regresión lineal, exponencial y análisis de estacionalidad.
* **Business Intelligence**: Transformación de datos históricos en insights operativos para planeación de demanda.
